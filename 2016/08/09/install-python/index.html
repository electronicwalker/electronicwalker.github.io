考虑到目前网上关于 Python 的代码以及教程以 Python2.x 居多，所以也描述了 Python2.x 的安装过程。

安装 Python

安装 Python2

运行下载到本地硬盘上的安装程序进行安装。

  • 64位安装程序名称：python-2.7.12.amd64.msi
  • 32位安装程序名称：python-2.7.12.msi

Python2 的默认安装路径在 C:\Python27

选中 Add python.exe to Path ，将 python.exe 的所在目录(C:\Python27)以及 python脚本目录(C:\Python27\Scripts)添加进系统变量 PATH，这样便可以直接在命令行执行 python.exe程序，以及 python脚本目录(C:\Python27\Scripts)下的所有程序，包括 easy_install.exe 和 pip.exe。

常见问题

如果我想同时安装 Python2.x 和 Python3.x，可以实现吗？
当然可以实现。
分别安装好 Python2.x 和 Python3.x，且两者的安装目录都已写入系统变量 PATH，因为 Python2.x 和 Python3.x 的程序名都是 python.exe，系统只是选择其中一个执行，故你需要分别修改两者的程序名为 python2.exe 和 python3.exe。
这样，你就可以在 cmd 或 powershell 通过 python2 调用 python2.x， 通过 python3 调用 python3.x。
总结一下操作步骤：

  1. 分别安装好 Python2.x 和 Python3.x，且确保两者的安装目录都已写入系统变量 PATH。
  2. 分别修改两者的程序名为 python2.exe 和 python3.exe。

    PS: 你也可以直接复制一个 python.exe 的副本，然后重命名，主要是为了防止重新使用安装程序进行卸载时发生错误。

  3. 在 cmd 或 powershell 通过 python2 调用 python2.x， 通过 python3 调用 python3.x
