反思学习方式——碎片式学习的陷阱

发表于 2016-08-12

学习方式，一直是影响个人学习的瓶颈，所以，我觉得有必要好好反思一下自己的学习方式。

你是否经历过…

你有没有听说过一个从未听过的“专业名词”，因害怕落后或跟不上主流而突然很想要学习？

于是你上网百度或订阅别人推荐的公众号，然后你像发现新大陆一样，沉浸在越来越牛的文章、越来越新颖的技术里，然后像海绵一样尽全力地去“吸收”知识。你订阅越来越多的公众号，收藏越来越多的文章，可是还是觉得不够。你下载越来越多的资料、文档存在硬盘里，可是这里面很多你可能都没有看过，也许再也不会看了。

没过多久，你就了解了各种各样的“专业名词”，可是你发现自己只是一个“专业名词”的搬运工，那些熟悉的“专业名词”事实上你根本讲不出所以然来，你感觉自己那些知识储备就像被残破的渔网网住的宝箱，不知什么时候就会被湍急的水流卷走。最后，你放弃了不断收集知识的过程，你觉得徒劳无功，兴趣索然，最后你觉得身体好像被掏空一样，什么都不想动了。

可没过多久，你又开始了上面的循环，宛若命运一般，明明想逃开，却又回到了最初的地方。

如果你也有过上述经历，那么，恭喜你，这篇文章对你而言并不是毫无价值。

如果你并没有上述经历，那么，恭喜你，不用浪费时间听我给你讲没有用的。

这都是我的错吗？

那么问题来了，罹患“焦虑症”、“松鼠症”和“懒癌”这三大绝症是我的错吗？

告诉你一件高兴的事，这一切都不你的错。要怪，就怪这个信息巨量化、知识碎片化的时代吧。

信息爆炸的互联网时代，让我们可以通过更简单与快捷的途径（搜索引擎）去获取海量的信息，但我们的获取的信息都是碎片化的，这些信息之间的关联性、组织性是极弱的。我们很难将碎片化的信息组织成一定的结构化数据，所以我们很容易就陷入盲目收集知识碎片，却不知道如何组织利用这些知识碎片来构建自己的知识体系。

{ %cq } 即使你拿回来一百万个知识的砂砾，也只是沙漠，人们爱看的，是沙子聚合成的金字塔。 { %endcq }

什么是(构建)知识体系？

所谓知识体系，是知识的组织与管理，将自己获取并加工的知识分门别类地储存在自己的硬盘或大脑中，以便于应用知识。

这里提到几点：

  • 知识需要进行加工
  • 知识需要分类管理
  • 知识需要被应用

怎么构建知识体系？

明确自己的知识需求

知识是无穷无尽的，我们的知识体系不需要包括所有的知识，只需要包括自己需要的知识，所以，必须明确自己需要的知识有哪些。

另外还必须记住一点，知识是会更新的，知识的需求也是会更新的。不再被应用的知识是过期的知识，过期的知识就要及时清理与更新。

获取并加工知识

要根据自己的需求去获取知识，避免上文提到的 知识收集癖。

获取知识的途径根据个人习惯有所不同，一般包括：

  • 人际关系
  • 搜索引擎
  • 书籍
  • 信息聚合平台
  • 博客
  • 慕课

获取的知识往往是粗糙的，可能不适应个人的理解与利用。所以我们需要对原生(Raw)的知识进行加工，也就是 深入思考，理解知识并将其转化成可以为己所用的知识。

运用知识

最关键的一步，就是运用所学的知识去解决实际的问题。知识学来不是为了显摆或收藏的，而是为了在实际中运用。

实际运用知识，才会让你了解知识 精妙的原理 与 创新的组合方式。

整理与保存知识

在得到属于自己的知识之后，你需要对自己的知识进行管理， 在这里拓展一个专有名词 Personal Knowledge Management (个人知识管理)，简称 PKM。

知识管理的方法有很多种，依据不同人的喜好与习惯进行选择即可。

交流与分享知识

浩瀚的知识要被人永久记忆是一件很困难的事，但也不是没有办法，我们说的办法当然不是指死记硬背了，而是交流与分享。

与人交流知识的过程中，我们会对知识的印象与理解都会随着有效的交流而逐渐加深。

另一种更高效的方式是分享知识，所谓分享，亦可说是传授，是自己提炼知识，将个人理解与体悟等隐性知识具象化语言、PPT、文档等显性知识的过程。

很简单，你教别人的过程中，学到最多的无疑还是你自己。

输入

知识碎片化的时代里，每个人都要了解自身的知识需求，不会被你用上的知识，都是没有必要收集的。不要做盲目的“松鼠”，收集一堆用不上的“松果”。

格式化输入

获取的知识需要深入的思考，而不是简单的收藏。

输出

知识需要运用于实际，实际运用才是知识的目的。

格式化输出

知识需要分享与交流，加深对整个知识体系的把握。

保存

知识需要有组织的管理方法，避免知识结构混乱化，建立清晰的知识体系。
