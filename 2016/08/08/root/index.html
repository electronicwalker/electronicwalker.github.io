溯源还是出走

发表于 2016-08-08

总在忙着赶路和细赏沿途风景的矛盾中败下阵来，最终得无偿失。

折腾了几天的时间，简单地实现了在github上构建了自己的博客的想法，然后大张旗鼓地开始筹备把整个建站的过程写成自己的第一篇博文。为了不让第一篇博客太随便，于是煞有介事地分析整个建站过程，打算详尽描述每个步骤，尽可能地让第一篇博客不太掉份。

分析完流程，开始描述第一个步骤：注册 github 账号。为了凸显建站的主要流程，本来不想多花时间在前期准备工作上，但是逃不过心里的那点强迫症，还是想找一个简洁但全面的解决方案。最终，想到了用一个 git 动态图片来展示注册过程，但是我目前手头上并没有什么现成的 gif 录制工具，于是打开了百度，开始搜索。

几十分钟后，我安装好了 LICEcap 工具，打算开始录制注册过程，才发现得再找一个邮箱，于是找了个QQ小号(●’◡’●)。好，正式开始录制了，在录制完第一个步骤之后，由于对新安装的 LICEcap 工具不够了解（其实也没啥功能），对录制结果不满意，本想着重新录制的，没想到注册已经完成了。

想想，觉得初期准备不够，也不满意 LICEcap 的录制功能，就又上百度找了款功能更强大的开源工具 ScreenToGif。

这样零零总总地折腾下来，总是录好一个样图，可打算在博文里引用图片时，又发现一堆问题，于是又是几个小时的百度时间。

时间过去地很快，问题也层出不穷，往往为了解决一个问题而浪费大量的时间在搜索答案上，更严重的是自己跟最初的目的背道相驰。

一开始萌生想写博客的想法，只是单纯出于想锻炼自己已经逐渐退化的表达能力，但不悦于平台博客的不可定制与界面的不友好，才打算建立个人博客，却也着实花了时间去了解与比较个人博客的建立方式。断断续续折腾到现在，好不容易个人博客搭建起来了，却没有了当初想要写文的热切了。

本为了体验更幽深的风景选择了小径，却不想被路上的荆棘划得遍体鳞伤，最初的想要到达目的地的热忱心情，也开始慢慢变淡。

世界那么大，我想去看看

世界太大了，总想去看看，却在没看过的风景中迷失自己的本来要到达的目的地。

不知为何，我突然想起了《西游记》里的唐僧和悟空。唐僧要去西天取经必须脚踏实地，一步一步地走过去，他不知道西天具体在在哪，但他知道下一步在哪。而悟空则不同，对于唐僧而言远在天边的西天，只是一个跟斗的功夫就可以到了。西天在悟空的眼里就是如来老儿住的地方，串门如同家常便饭，哪需要花个十年八载的时间一路走过去。而悟空时时都可以翻个筋斗到西天，却不得不脚踏实地地陪唐僧走尽这山水洞天，这才是悟空的修行——忍耐。

迫使悟空踏上修行之路的无疑是紧箍咒，但于我而言，却是脚踏实地走一程的魔咒。我像悟空一样，一开始就知道了西天的实际位置，也知道一步即达的方法，却羡慕唐僧一路走过去的充实而打算一步一步走过去。然而这种半吊子的魔咒毕竟不比紧箍咒的来场大，禁不起挣扎，最终没修得忍耐，也没到达西天。

如果那么想看沿途的风景，那就别老想着西天了。
