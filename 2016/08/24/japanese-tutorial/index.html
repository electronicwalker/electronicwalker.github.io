日语五十音入门

发表于 2016-08-24

最近花了几天时间学习了日语五十音，满足了自己一直以来想认真学习一下日语的‘冲动’。说来惭愧，做了这么多年的 ACG 文化爱好者，却一直都没有真正了解过日语，不得不说是个不小的遗憾。所以这次我终于不满足于做那个以为罗马音是日语音标的语痴了。

初识日语

任何一门成熟的语言一般都会有一套口语系统和一套文字系统。而且，一般来说，都是先有口语，再有文字。这里就稍稍讲一点日语的发展历史，当然，没兴趣的同学就别勉强自己看完了。

汉字的引入

日语也是如此，在很长一段时间里，日语是没有文字系统的。直到中国唐代时期，中国汉字传入了日本，日本便用汉字来作为文字系统来表示口语的意思，也就是作为表意符号。但你想想啊，唐朝时期的汉字可不是现在的中文简体，而是文言文，学过文言文的同学都应该知道文言文有啥特点——字词的意义太晦涩。我猜唐朝时期的文人应该也不会占多大的比例，那么可想而知，在采用汉字作为表意符号的日本，除非你愿意苦读十年寒窗或者愿意花钱请家教，不然你肯定学不会汉语的。

假名的出现

既然汉字在日本水土不服，那没办法，要么弃用，要么就改喽。怎么改呢？上文说到，汉字的意思太晦涩了，不好理解，那么就别用它来表示口语的意思了，用来表示口语的读音好了。于是，假名就出现了。所谓假嘛，学过文言文的都应该可以理解，就是借的意思，这里就是借用了汉字来表示日语口语的意思。而最开始流行的假名据我所知是万叶假名，这时候，汉字不再是作为表意符号，而是作为表音符号。

平假名的出现

万叶假名起初主要流行于教育程度相对较低的宫廷女官之间，由于她们常年使用万叶假名的汉字，故简化了汉字的书写笔画，使其变成了类似中国草书的字体，称之为草假名，最终草假名的字形被固定下来，也就形成了平假名。

片假名的出现

虽然平假名已经出现，但当时日本的官方正式文本记录还是使用纯汉字。所以宫廷子弟也好，大学考生也好，都是要学习汉语的。但是由于纯汉字的表记方式与日语在语序读音上不对应，所以这些‘精明人’就在汉文旁边添加种种助词与记号，如果你有过用汉字注解英文的经历的话，你可能会更好理解了(●’◡’●)。这些助记符号逐渐发展成了乎古止点，再进一步发展，乎古止点也被进一步简化，最终形成了片假名。

概念介绍

现在进入正文，在讲五十音之前，得先讲些语言学的概念。

音素 ( Phoneme )

音素是最小的语音单位。

不同语言（包括方言）的音素是不一样的。

在汉语中，音素指的是 23 个声母和 39 个韵母。

在英语中，音素指的是 20 个元音和 28 个辅音。

在日语中，音素指的是 5 个元音和 19 个辅音（包括变形）。

音节 ( Syllable )

音节是由音素组成的基本语音单位。

在汉语中，音节通常指的是一个汉字的读音（拼音），而一个汉字的读音由一个声母和一个韵母组成。

在英语中，音节通常可以是一个元音音素，也可以是一个元音音素和若干辅音音素的组合，也可以是双元音音素的组合。

在日语中，音节一般对应一个假名(拗音是两个假名对应一个音节)。在日语中，音节也被称为拍。

假名 ( かな / Kana )

假名是日语的表音文字（音节文字）。

假名分为平假名和片假名。

平假名和片假名具有一一对应的关系。

现代日语中平假名、片假名和汉字混用。

平假名用来注解外来词汇，一般指欧美词汇。

罗马字 ( ロマジ / Romaji )

罗马字是用拉丁字母来表记日语的一种方法。

主流的有 训令式罗马字 和 平文式罗马字。

平文式罗马字

平文式罗马字的拼写基于英语音韵学。因此对不熟悉日语的人而言，平文式罗马字的发音将更接近日语发音。

转写规则

助词

  • 当“he”へ 用作助词时，写作“e”。
  • 当“ha”は 用作助词时，写作“wa”。
  • 当“wo”を 用作助词时，写作“o”。

长音

在传统平文式和修订平文式中：

  • 长音“o”和“u”使用长音符号表示——如：长音“o”写作“ō”。
  • 长音“e”写作“ei”。
  • 长音“i”写作“ii”。
  • 外来词汇中，所有长音都用长音符号表示。

在改良平文式中：

  • 所有长音使用双写元音表示。

拨音

在传统平文式中：

  • “n”（ん）在辅音前写作“n”，但在唇音（b, m, p）前写作“m”。在元音和“y”之前，有时写作“n-”，以避免混淆。
    • 案内（あんない）：annai
    • 群馬（ぐんま）：gumma
    • 簡易（かんい）：kan-i
    • 信用（しんよう）：shin-yō

在修订平文式中：

  • 全部写作“n”，元音前使用隔音号。
    • 案内（あんない）：annai
    • 群馬（ぐんま）：gunma
    • 禁煙（きんえん）：kin'en

在改良平文式中：

  • 全部写作“n̄”（n加上长音符号），避免了使用隔音号，但在计算机中，有些字体可能无法正常显示。
    • 案内（あんない）：an̄nai
    • 群馬（ぐんま）：gun̄ma
    • 禁煙（きんえん）：kin̄en̄

促音

在平文式罗马字中，双写后续假名的辅音表示促音っ，如后续假名的辅音有两个字母的，变换规则如下：sh → ssh, ch → tch, ts → tts。

  • 結果（けっか）：kekka
  • さっさと：sassato
  • ずっと：zutto
  • 切符（きっぷ）：kippu
  • 雑誌（ざっし）：zasshi
  • 一緒（いっしょ）：issho
  • こっち：kotchi
  • 抹茶（まっちゃ）：matcha
  • 三つ（みっつ）：mittsu

变种

平文式罗马字有许多变种，主要和长音的表示有关。在“東京（とうきょう）”一词中，长音“ō”分别写作：

  • Tōkyō —— 使用长音符号。即遵照传统平文式，最为常用。
  • Tokyo —— 不使用任何记号。日语词吸收进英语时使用。在英文文章中是事实上的标准。
  • Tôkyô —— 使用抑扬符号（^）。在字处理器不支持长音符号会使用，但随着Unicode的普及变得少见。
  • Tohkyoh —— 长音“o”表示为“oh”，有时也叫做“护照平文式”，在护照中可以使用这种写法。[1]
  • Toukyou —— 直接按照假名逐字转写，区别“おう”（如：とうきょう（東京）→Toukyou）和“おお”（とおい（遠い）→tooi）。这种规则来自罗马字输入法，在不方便输入长音符号时很常用。
  • Tookyoo —— 双写元音，即遵照改良平文式，在拼写外国人名时也很常用。

音节分类

清音

清音（指该音节的辅音）发音时声带不振动。

浊音

浊音（指该音节的辅音）发音时声带振动。

浊音假名的写法是在清音假名上加两点。

半浊音假名的写法是在清音假名上加一句号。

拨音

拨音是鼻音，但是发音方法不一。

拨音 ん 的读音根据后一个假名而变化。

拗音

拗音是指い 段假名（含浊音、半浊音）（除い外）「き し ち に ひ み り ぎ じ ぢ び ぴ」和复元音「や、ゆ、よ」拼起来的音节，共有三十六个，在「い」段假名后面右下角加小写的「や、ゆ、よ」来表示。

拗音类似于汉语拼音。

语音规则

促音

促音（日本语：そくおん sokuon）是在日语中用来表示停顿的符号，写法为っ、片假名ッ（注意这里是小写）。

促音不发音，但要占一个音节，即占一拍。

长音

长音是把元音延长一拍的发音。
