腾讯视频下载地址解析

发表于 2016-09-16

今天我们来解析一下腾讯视频的真实下载地址。

先声明一下，我这次解析所用的环境。

准备环境

  • 浏览器：chrome 浏览器

其实，任何一个浏览器都可以，只是在这里，我主要使用 chrome 浏览器的 Developer Tools 来抓取数据包，你也可以替换成其他的抓包工具。

好，现在进行实际操作。

1.打开任意一个视频播放页面，此处示例是 http://v.qq.com/cover/4/4gmobofw9fltve9.html。

2.按下 F12 打开 Developer Tools，选中 Network 选项卡，并点击 Other 过滤标签。

3.切回网页界面，按下 F5 刷新页面。

4.再切回 Developer Tools，此时 Developer Tools 将会刷新内容，找到 getvkey 项。

5.点击 getvkey 项，选中 Preview，点击三角，展开内容。

6.分析内容。

  • filename 标签内是未分段的视频文件名
  • keyid 标签内是分段视频文件名
  • key 标签内是分段视频的 key 值，与 keyid 一一对应

7.由上面三个内容我们可以拼凑成一个视频地址。

  • 除去紫框圈中的「.1」,也就是「d0015pu5010.p201.mp4」，这是 filename 标签内包含的内容，而「.1」是 \<keyid 标签内包含的内容的尾缀，表示第 1 段视频。

  • 而绿框圈中的「?」表示后面为传递的参数。

  • 红框圈中的部分即为 key 标签包含的内容。

8.我们还有知道服务器地址，才能访问到视频，这里我就直接贴出来了。

  • http://117.21.186.[11-26]/vipzj.video.tc.qq.com/

  • 上面中括号里面的 [11-26] 表示可选数字，数字范围为 11~26，包括11和26。

9.拼凑成完整的视频地址。

  • http://117.21.186.12/vipzj.video.tc.qq.com/d0015pu5010.p201.1.mp4?vkey=64B1129502A2ADA15121F40155F1C3FF3E9EA9C8694BB24441C7D359B389CF9DE85584E6ACE80A8460FF21E3E5BAE3837B4888C8B5C52E12E60236A48D0B6BE654C0B2BF0987D90C2E27D686E0A0B18A333315A9855BCCABEB9E32FA329E715A22BED9B7DFC1E4AB4CF9A985A02D85D144A2A5E085594F3D

你可以在浏览器里打开这个地址，这里你就可以看到第一段视频可以播放了。

这时候，你可以在视频上，右键，另存为。也可以在任意下载器（比如，迅雷、旋风、IDM等等）里新建下载，把视频地址粘上就好。

当然，你也发现了，这个视频只有一小段，并不完整。这是因为目前视频网站为了降低数据传输的时延，把一个完整的视频分成了一个个分段。

另外，视频加载时只会每次加载一个片段，等你快看完这个片段时，再缓冲下一个视频片段。

所以，如果你想获取一个视频的所有片段，你可能得依次点击播放进度条，让网页把每个片段缓冲下来，这样你依然可以通过以上方式获取所有的视频片断下载地址。这里要注意一点，一个 getvkey 文件对应一个视频片段。

当你把所有的视频片断下载下来之后，你可能想将这些视频合并。事实上，你可以找到很多视频合并工具，我使用的是 My MP4Box GUI。

好了，今天的教程就先到这吧，有什么问题请给我发 email（541488630@qq.com）。
